## Quick Start Guide

Get up and running with Aagnia services in minutes. Follow these steps to explore documentation and begin your project.

<Steps>

<Step title="Review Documentation" icon="book-open">

Browse sections like Quickstart and Authentication. Understand core concepts before diving in.

</Step>

<Step title="Set Up API Access" icon="key">

Obtain your API key from the dashboard at `https://dashboard.example.com`. Use it for all requests.

````javascript
const apiKey = `YOUR_API_KEY`;
const response = await fetch(`https://api.example.com/projects`, {
  headers: { Authorization: `Bearer ${apiKey}` }
});
````

</Step>

<Step title="Build Your First Project" icon="code">

Create a sample project using the API. Test endpoints for web, IoT, or AI services.

<CodeGroup tabs="JavaScript,Python">

````javascript
// Create a new project
const projectData = { name: "My IoT App", type: "iot" };
const newProject = await fetch(`https://api.example.com/projects`, {
  method: "POST",
  headers: { "Content-Type": "application/json", Authorization: `Bearer ${apiKey}` },
  body: JSON.stringify(projectData)
});
````

````python
import requests

api_key = "YOUR_API_KEY"
project_data = {"name": "My IoT App", "type": "iot"}
response = requests.post(
    "https://api.example.com/projects",
    json=project_data,
    headers={"Authorization": f"Bearer {api_key}"}
)
````

</CodeGroup>

</Step>

<Step title="Deploy and Monitor" icon="rocket">

Launch your solution and use dashboards for insights. Integrate webhooks for real-time updates.

</Step>

</Steps>
